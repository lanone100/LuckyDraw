抽籤公式說明
① 完全隨機公式
Math.random() × N → floor
// N = 名單總人數
index = Math.floor(Math.random() × N)
winner = candidates[index]
② Fisher-Yates 洗牌演算法
// 抽多人時保證不重複
for (i = N-1; i > 0; i--) {
j = floor(random × (i+1))
[arr[i], arr[j]] = [arr[j], arr[i]]
}
winners = arr.slice(0, K)
③ 加權機率公式
// w[i] = 個人權重
totalW = Σ w[i]
rand = random × totalW
cumW = 0
for each candidate:
cumW += w[i]
if rand ≤ cumW → selected!
④ 期望中獎率計算
P(中獎) = K / N
// K = 抽出名額, N = 總人數

期望值 E = K × (1/N)
不中獎率 = 1 - (K/N)
